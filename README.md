# Tic-Tac-Toe-C
A console-based Tic Tac Toe game built using C programming.
---

## 📌 Features

- 3x3 game board using 2D array
- Two-player mode (X and O)
- Input validation
- Win detection (rows, columns, diagonals)
- Draw detection
- Turn switching logic

---

## 🛠 Concepts Used

- 2D Arrays
- Loops
- Conditional Statements
- Functions
- Game Logic Implementation
- Input Validation

---

## 🎮 How to Play

1. The board is displayed with empty spaces.
2. Players take turns entering row and column values.
3. The first player to align three symbols (row, column, or diagonal) wins.
4. If all 9 moves are used and no one wins, the game ends in a draw.

---

## ▶️ How to Run

Compile:
```
gcc main.c -o tictactoe
```

Run:
```
./tictactoe
```
